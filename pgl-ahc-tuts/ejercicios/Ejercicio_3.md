Ejercicio 3

Se modificó el "_layout" del drawer con la intención de que solo apareciera la pantalla de login hasta no verificar que hay token. Para ello nos valimos de variables de estado y la función grande. Cada vez que se inicia la pantalla se renderiza primero la pantalla de carga con la variable "loading" porque eso se renderiza antes de que se active la función, utilicé un useEffect para afectar al código y no al renderizado, como useEffect no puede ser asíncrono utilizo una función anónima que afecta al router y se inicia de manera instantánea.
Dentro del useEffect llama a la función asíncrona del tokenService para traer el token, luego hace doble verificación, si trae token será string, sinó no valdrá, por último si ha concluido la extracción del token, guarda el valor del token que si existe en "setHasToken" y lo actualiza, actualiza el valor que controla si está cargando y pasa a ser falsa, luego hay una última verificación en la que si la variable actualizada no es el token, retorna nulo y no se renderiza lo demás ya que renderiza nulo, en caso de que el token esté, se logueará y te redigiriá a la pantalla welcome y permitirá que el drawer se renderice.
